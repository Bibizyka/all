Задания по лабораторным работам общего курса "Аппаратные средства вычислительной техники", 2-курс, 1-й семестр

Всего предусматривается выполнение 4 работ, максимальная оценка каждой работы 6 баллов(максимальная сумма 24 балла). Каждая работа оформляется в виде реферата, где до ответов на вопросы необходимо в виде преамбулы привести сведения об условиях экплуатации и технических свойствах оборудования.

  1. Описание компьютера
    • комплектность компьютера
    • материнская плата
    • проект локальной сети
    • необходимое оборудование для работы в беспроводной сети
  2. Разработка программы для Web-камеры
    • подбор оборудования
    • выбор необходимых функций
    • блок-схема и интерфейс
    • демонстрация работы программы и документирование
  3. Разработка коммуникационной программы для микроконтроллера
    • подбор оборудования
    • выбор протокола обмена для реализации
    • интерфейс и доступные элементы управления
    • демонстрация работы программы и документирование
  4. Разработка программы для контроллера
    • возможности ПЛК150-220.И-М фирмы ОВЕН
    • освоение среды CoDeSys по тестовому примеру
    • разработка программы автоматизации
    • демонстрация работы программы и документирование
